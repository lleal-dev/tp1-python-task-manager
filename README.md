## 🇧🇷 Português

### Sobre o projeto

Este projeto é um gerenciador de tarefas desenvolvido em Python como trabalho acadêmico, com o objetivo de praticar conceitos fundamentais da linguagem, como variáveis, listas, dicionários, funções e estruturas de repetição (`for` e `while`).

O programa é executado no terminal e permite ao usuário gerenciar uma lista de tarefas de forma interativa por meio de um menu de opções.

### Funcionalidades

- **Adicionar Tarefa** — permite cadastrar uma nova tarefa com título, descrição e prazo
- **Listar Tarefas** — exibe todas as tarefas cadastradas, enumeradas e organizadas
- **Marcar como Concluída** — remove a tarefa da lista ao marcá-la como concluída
- **Excluir Tarefa** — remove uma tarefa da lista permanentemente
- **Sair** — encerra o programa

### Conceitos praticados

| Conceito             | Aplicação no projeto                                   |
| -------------------- | ------------------------------------------------------ |
| Variáveis e tipos    | Armazenamento de título, descrição e prazo das tarefas |
| Listas (`list`)      | Estrutura principal para armazenar as tarefas          |
| Dicionários (`dict`) | Cada tarefa é representada como um dicionário          |
| Funções              | Uma função dedicada para cada funcionalidade           |
| DocStrings           | Documentação de cada função                            |
| Parâmetros padrão    | Valores default nos parâmetros das funções             |
| `while`              | Loop do menu principal, repete até o usuário sair      |
| `for` + `enumerate`  | Listagem enumerada das tarefas                         |

### Como executar

```bash
python main.py
```

## 🇺🇸 English

### About

This project is a command-line task manager built with Python as an academic assignment. The goal is to practice core programming concepts such as variables, lists, dictionaries, functions, and loop structures (`for` and `while`).

The program runs in the terminal and allows the user to manage a task list interactively through a menu-driven interface.

### Features

- **Add Task** — register a new task with a title, description, and deadline
- **List Tasks** — display all registered tasks, numbered and organized
- **Mark as Completed** — remove the task from the list upon completion
- **Delete Task** — permanently remove a task from the list
- **Exit** — close the program

### Concepts practiced

| Concept               | Application in the project                    |
| --------------------- | --------------------------------------------- |
| Variables & types     | Storing task title, description, and deadline |
| Lists (`list`)        | Main data structure for storing tasks         |
| Dictionaries (`dict`) | Each task is represented as a dictionary      |
| Functions             | One dedicated function per feature            |
| DocStrings            | Documentation for each function               |
| Default parameters    | Default values in function parameters         |
| `while`               | Main menu loop, repeats until the user exits  |
| `for` + `enumerate`   | Numbered task listing                         |

### How to run

```bash
python main.py
```
